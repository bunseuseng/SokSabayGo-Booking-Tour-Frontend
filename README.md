# 🌏 Soksabay Go — TukTuk Booking Platform

## 📌 Project Overview

Soksabay Go is a modern web application that allows tourists to explore and book TukTuk tours in Cambodia.
The platform provides a smooth experience for discovering tours, selecting drivers, and managing bookings.

---

## 🚀 Tech Stack

This project is built with:

* ⚡ Vite
* ⚛️ React (TypeScript)
* 🎨 Tailwind CSS
* 🧩 shadcn/ui

---

## 💻 Getting Started

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project folder
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```
